¿Qué es un alert en HTML?

Un alert en HTML es una ventana emergente que se utiliza para mostrar información al usuario de forma modal, es decir, el usuario no puede interactuar con el resto de la página hasta que haya cerrado la ventana emergente.

¿Para qué sirve un alert?

Los alerts se utilizan para:

· Mostrar mensajes de error: Se pueden usar para informar al usuario de que ha habido un error al realizar una acción.

· Mostrar mensajes de confirmación: Se pueden usar para confirmar que el usuario desea realizar una acción que podría tener consecuencias importantes.

·Mostrar información importante: Se pueden usar para mostrar información importante al usuario, como un anuncio o una noticia.

¿Cómo se crea un alert en HTML?

Un alert se crea usando la función window.alert(). Esta función toma un único argumento, que es el mensaje que se desea mostrar al usuario.

Ejemplo de código.

            
           <script>
           window.alert("¡Hola mundo!");
           </script>
