ARIA atributes

  • ROLE - Para todo el conjunto se crea un elemento con el rol de 'radiogroup', para cada boton se usa el rol 'radio'
  • ARIA-CHECKED - Funciona para decir si el botón está check desde su inicio (se puede usar JS para modificar el valor)
  • ARIA-LABELLEDBY | ARIA-LABEL - Dice el valor (texto del botón) que tiene el elemento
  • ARIA-DESCRIBEDBY - Descripción del radiogroup
  • ARIA-REQUIRED - Menciona si es obligatorio seleccionar uno de los elementos
